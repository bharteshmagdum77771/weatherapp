# Weather App
A simple and responsive weather application built with React.js. Get real-time weather updates for any city with a clean and user-friendly interface.

## Features
Search for any city and view current weather
Displays temperature, wind speed, and weather conditions
Responsive design for mobile and desktop
Easy-to-read weather icons and information

## Technologies Used
React.js,
html,
javascript,
CSS/Bootstrap,
Vite,
OpenWeatherMap API
